Instructions for print

  • Check the checkbox to the left of perks that you want to print. Also check only the categories(s) you wish to print.
  • Make sure you print in landscape
  • For best effects, print with backgrounds, to get the best design contrast and readability.
  • Each character can have a maximum of 8 perks from any category.
